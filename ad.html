No walled garden

Local doesn't mean isolated.

SegSmart reads from the systems you already run and hands its results to whatever you send mail with — in open formats, through integration points you own. There's nothing proprietary to be locked into, and no export fee to leave.

Reads from

your existing data, in place
  • →CSV / TSV / Excel exports — any encoding, any delimiter, missing columns and all
  • →MySQL · MariaDB · PostgreSQL · SQLite — read-only, never a write
  • →Google BigQuery
  • →Shoptet API — and the same pattern fits WooCommerce, PrestaShop, Magento

Hands off to

your mailer, your way
  • →An import-ready recipient file (e-mail + name) any ESP accepts
  • →A webhook to n8n / Zapier / your own script
  • →Plain JSON + CSV — no proprietary format, no lock-in
  • →No mailer at all? Download the file and send it yourself
